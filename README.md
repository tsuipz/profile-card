# Profile Card

## About

Here is Patrick Tsui's Profile Card. A paperless way of sharing social media and contact information.

- Card contains
  - Name
  - Job Title
  - Skills
  - Social Media Sites
  - QR Code

## How to run locally

1. Clone and fork this repo
2. Run `npm install` to load dependencies
3. Run `npm start`
4. Go to http://localhost:3000/
